# Student Database Management System

A simple **Student Database Management System** built using **Node.js, Express.js, and HTML/CSS/JavaScript**.  
This project allows administrators to manage student records efficiently with basic CRUD operations.

---

## 📌 Features

- Add new student records
- View all students
- Update student details
- Delete student records
- Separate Admin and Student routes
- Simple and clean UI
- RESTful API structure

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** (File-based / JSON / MongoDB – as per implementation)
- **Tools:** Git, GitHub
